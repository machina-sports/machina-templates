# Sports Narrator — Guia de Uso / Usage Guide

Invocação / Invocation:
- **en-US:** "Alexa, open **sports narrator**"
- **pt-BR:** "Alexa, abrir **narrador esportivo**"

---

## Futebol / Soccer (Brasileirao Serie A)

| Portugues (pt-BR) | English (en-US) |
|---|---|
| "resultados do brasileirao" | "soccer scores" |
| "placar do futebol hoje" | "soccer scores today" |
| "quais foram os placares do futebol" | "what were the soccer scores" |
| "me diga os resultados do futebol" | "tell me the soccer results" |
| "jogos de hoje" | "football scores" |

## Salvar time favorito / Save favorite team

| Portugues (pt-BR) | English (en-US) |
|---|---|
| "meu time favorito e o Flamengo" | "my favorite team is Flamengo" |
| "eu torco para o Corinthians" | "I follow the Chiefs" |
| "eu sou fa do Palmeiras" | "I'm a Lakers fan" |
| "meu time e o Santos" | "my team is the Cowboys" |

Times de futebol suportados / Supported soccer teams:
Flamengo, Palmeiras, Corinthians, Sao Paulo, Santos, Gremio, Internacional, Atletico Mineiro, Cruzeiro, Botafogo, Vasco, Fluminense, Bahia, Fortaleza, Athletico Paranaense, Bragantino, Vitoria, Juventude, Ceara, Criciuma

Times de NFL suportados / Supported NFL teams:
Kansas City Chiefs, Dallas Cowboys, New England Patriots

Times de NBA suportados / Supported NBA teams:
Los Angeles Lakers, Golden State Warriors, Boston Celtics

## Notas

- O Brasileirao retorna os ultimos 7 jogos com status `closed` ou `ended`
- A NFL retorna os ultimos 7 jogos com status `closed` ou `complete` (temporada atual calculada automaticamente)
- A NBA retorna os ultimos 7 jogos com status `closed` ou `complete`
- A atualizacao personalizada so busca dados dos esportes que o usuario acompanha
- Respostas em pt-BR quando o dispositivo esta configurado para Portuguese (Brazil)
