# Corinthians Fan Engagement Skill

This skill provides a suite of tools for generating engaging content for Corinthians fans.

## Features

- **Match Previews:** Generate detailed match previews for upcoming Corinthians games.
- **Match Summaries:** Create summaries of completed matches, including key events and analysis.
- **Social Media Content:** Generate social media posts to engage with fans before, during, and after matches.

## Workflows

### `generate-match-preview`

This workflow generates a match preview for an upcoming Corinthians game.

**Inputs:**

- `home_team`: The home team.
- `away_team`: The away team.
- `match_date`: The date of the match.
- `match_venue`: The venue of the match.
- `competition`: The competition.

**Outputs:**

- `match_preview`: The generated match preview.
